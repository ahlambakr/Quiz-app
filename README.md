# About
- Backend application demo using Spring Boot & postgreSQL & Redis
# Tech Stack 
 - PostgreSQL & pgAdmin
 - Java & Spring boot
 - Redis
 - Dependencies used:
 - Spring web
 - JDBC API
 - Spring cache
 - Spring Data Redis
 - Spring Data API
 - PostgresSQL Driver
# installation
1. Clone Repository and open project in IDE
2. Set up database
3. set up redis 
4. Run application
